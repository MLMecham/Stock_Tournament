## How It Works

Six profiles compete weekly to pick three stocks each from an 80-ticker S&P 500 universe spanning tech, financials, healthcare, energy, and consumer staples. Every Monday, each profile calls `pick()` and the selections are logged with entry prices. After seven days, exit prices are fetched, returns are calculated, and the results feed a training set. Rick RF is the only profile that learns — he trains a Random Forest classifier on accumulated outcomes and improves his picks over time. All data is stored as Snappy-compressed Parquet files partitioned by year and week; the leaderboard and this README are regenerated automatically via GitHub Actions.
